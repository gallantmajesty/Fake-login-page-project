# GitHub-Style Login Demo

Professional-looking GitHub login page for demonstration purposes.

## Features
- Realistic GitHub dark theme
- Password visibility toggle
- Credential capture (logged in Vercel)
- Smooth animations and responsive design
- Fake successful login flow

## Deployment on Vercel
1. Go to [vercel.com](https://vercel.com)
2. Create new project
3. Upload or connect this folder
4. Deploy

After deployment, visit the URL and try logging in. Check **Vercel Dashboard → Logs** to see captured credentials.

**Note**: This is for educational / demonstration / red teaming use only.
